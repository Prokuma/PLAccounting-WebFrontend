# PLAccounting-WebFrontend
自作会計ソフトウェア「PLAccounting」のウェブフロントエンド。

## 使用言語・フレームワーク
- TypeScript
- Svelte/SvelteKit
- Carbon Components Svelte

## デバッグ環境
依存パッケージ導入
```bash
`npm install`
```

実行
```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## ビルド
ビルド
```bash
npm run build
```
